### Pre-order 遍历

顺序遍历是首先访问根。然后遍历左子树。最后，遍历右边的子树。

### In-order 遍历

有序遍历是首先遍历左子树。然后访问根。最后，遍历右边的子树。

### Post-order 遍历

后顺序遍历是首先遍历左侧子树。然后遍历右边的子树。最后，访问根
