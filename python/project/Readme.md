##  创建django 项目

```
// create project Django
django-admin startproject mysite 

// 在创建文件下运行服务器
python manage.py runserver

// 默认端口8080， 添加5050 可改变端口
python manage.py runserver 5050

python manage.py startapp name_app   // 创建主文件夹，存放应用代码的地方
```
